This project analyzes Google Ads campaign performance data to uncover actionable insights for marketing strategy. It involves data cleaning, exploratory data analysis (EDA), visualization, and interpretation to evaluate how ad spend translates into conversions and roi.
Methodology
Data Cleaning:
Remove duplicates, handle missing values, normalize text fields.
Standardize numerical columns 
Exploratory Data Analysis (EDA):
Univariate: Histograms, boxplots for spend, clicks, conversions.
Bivariate: Scatterplots (cost vs sales, clicks vs conversion rate).
Multivariate:Heat map, Scatterplot
Visualization:
Campaign-level comparisons with Seaborn palettes.
Insights & Recommendations:
Identify campaigns with high ROI.
Suggest budget reallocation to maximize efficiency.
